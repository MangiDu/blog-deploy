# 霓虹灯按钮

Neon button

hover 时的倒影使用了 -webkit-box-reflect （非标准属性），霓虹灯的雾气效果是 box-shadow 重叠使用得到的结果。

整体的色调改变使用的 css-filter: hue-rotate(0deg) 色相旋转。filter - MDN

# Typing-demo

This is a typing demo.
.typing-demo {
    animation: blink .5s step-end infinite alternate;
    white-space: nowrap;
    overflow: hidden;
    border-right: 2px solid;
    font-family: monospace;
    font-size: 16px;
}
@keyframes blink {
    50% {
        border-color: transparent
    }
}

animation :

  • animation-timing-function : 定义 CSS 动画在每一动画周期中执行的节奏
  • animation-direction : 指示动画是否反向播放

# 网点图

不使用filter

原图
网点图

关键代码：

$bg-0 = url(https://assets.codepen.io/2017/portrait_woman.jpg);
$d = 6px;
.noproc {
    .halftone {
        background: $bg-0 50% / cover, radial-gradient(circle, #000, #fff) 0 0/$d $d round;
        background-blend-mode: screen;
        filter: contrast(19);
    }
}

用 filter 处理图片：

$bg-1 = url(https://assets.codepen.io/2017/portrait_tiger.jpg);
$d = 6px;
.filter {
    .halftone {
        position: relative;
        background: radial-gradient(circle, #000, #fff) 0 0/$d $d round;
        filter: contrast(19);
        overflow: hidden;
        &::after {
            content: '';
            position: absolute;
            top: 0;
            right: 0;
            bottom: 0;
            left: 0;
            background: $bg-1 50% / cover;
            filter: blur(3px) brightness(1.15) grayscale(1);
            mix-blend-mode: screen;
        }
    }
}

blend-mode 可选的属性值见此：blend-mode

本示例中用到的 screen 值：最终的颜色是反转顶层颜色和底层颜色，将反转后的两个颜色相乘，再反转相加得到的和得到的结果。黑色层不会造成变化，白色层导致白色最终层。其效果类似于（被投影仪）投射到投影屏幕上的两个图像。【好难懂啊这个描述】
