# 字符串

# KMP 算法

Knuth–Morris–Pratt （KMP）算法是一种改进的字符串匹配算法，它的核心是利用匹配失败后的信息，尽量减少模式串与主串的匹配次数以达到快速匹配的目的。它的时间复杂度是 O (m + n)。

判断模式串是否是主串的子串，按照一般的想法，会从主串首个字符开始对模式串的所有字符进行循环匹配，渐进时间复杂度为 O (mn)。很多时间浪费在了重复判断上。

如果每次不匹配的时候，我们能立刻知道是哪个元素不匹配，下一步可以从哪里开始重新匹配，就可以简化匹配的过程，也是 KMP 算法的核心思想。
