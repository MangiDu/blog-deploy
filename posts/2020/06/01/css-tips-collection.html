# 文字背景图

IMAGE
IMAGE

mix-blend-mode 属性表明元素的内容如何与其父级元素的内容、该元素的背景混合。

使用 background-clip 也可以实现，该属性表明元素的背景在其边框盒、边距盒、内容盒之下如何展现。

/* 部分代码 */
.colorful-text-1 {
    background: url($img);
    .text {
        color: #000;
        mix-blend-mode: lighten;
        background-color: #fff;
    }
}
.colorful-text-2 {
    background: url($img);
    -webkit-background-clip: text;
}

# 第二种实现方式

translateZ(-1)
translateZ(-2)
translateZ(-3)
  • transform-style : 设置元素的子元素是位于 3D 空间中还是平面中。
  • perspective : 指定了观察者与 z=0 平面的距离，使具有三维位置变换的元素产生透视效果。 z>0 的三维元素比正常大，而 z<0 时则比正常小，大小程度由该属性的值决定。

总结就是父元素设置 transform-style: preserve-3d 和 perspective: 1px ，子元素设置不同的 transform: translateZ
