利用两个伪元素，在鼠标悬停时改变伪元素的宽高，以呈现出动态变化的效果，需要注意的是 transition 属性也可以设置延时效果。

图片需要先叠底色，再处理 mix-blend-mode :

    /* ... */
    background: url($img), #0ff;
    background-blend-mode: lighten;
    /* ... */
}
